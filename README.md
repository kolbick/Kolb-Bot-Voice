# Kolb-Bot Voice

An AI voice assistant you can talk to — it can see your screen, control your browser, run commands, read and write files, and more.

---

## How to use it

### Option 1 — Just use it in your browser (no download needed)

1. Go to the download page (ask whoever set this up for the link, it looks like `https://xxxxx.trycloudflare.com/download`)
2. You'll see a voice widget right on the page — click the microphone button and start talking

That's it. No install required.

> **On iPhone?** Open the page in Safari, tap the Share button at the bottom, then tap **Add to Home Screen**. It will look and feel like a real app on your phone.

---

### Option 2 — Download the desktop app (gives the AI full control of your PC)

The browser version is great for chatting, but if you want the AI to actually control your computer — take screenshots, open websites, run programs, read your files — you need the desktop app.

#### Step 1 — Download it

Go to the download page and click the button for your computer:

- **Windows** → click **Download for Windows**
- **Mac** → click **Download for macOS**
- **Linux** → click **Download for Linux**

#### Step 2 — Install it

**Windows:**
1. Open the `.exe` file you downloaded
2. If Windows shows a warning saying "Windows protected your PC", click **More info** → **Run anyway** (this is normal for apps that aren't from the Microsoft Store)
3. Follow the installer

**Mac:**
1. Open the `.dmg` file you downloaded
2. Drag the **Kolb-Bot Voice** icon into your **Applications** folder
3. The first time you open it, right-click the app → **Open** → **Open** again (Mac asks this once for apps downloaded from the internet)

**Linux:**
1. Open a terminal in the folder where you downloaded the file
2. Run: `chmod +x "Kolb-Bot Voice-*.AppImage"`
3. Then double-click the file, or run it from the terminal

#### Step 3 — Set it up (first launch only)

When you open the app for the first time, a **Settings** screen will appear asking for a **Relay URL**.

- Go back to the download page
- You'll see a box that says **"Your relay URL"** — click it to copy the address
- Paste it into the Settings box in the app
- Click **Save & Reconnect**

You only have to do this once.

#### Step 4 — Talk to it

Press the big **Talk** button and start speaking. The AI can hear you and will talk back.

---

## What can it do?

Once the desktop app is connected, you can ask it things like:

- *"Take a screenshot and tell me what's on my screen"*
- *"Open Google and search for the weather"*
- *"What files are in my Downloads folder?"*
- *"Copy this text to my clipboard: ..."*
- *"What time is it?"*
- *"Run a speed test"*

## Troubleshooting

**The app says "No Tools Connected" or "Tools Disconnected"**
The relay URL may have changed (it changes every time the server restarts). Go back to the download page, copy the new relay URL, open the app's settings (⚙ gear icon in the top right), paste it in, and click Save.

**The download buttons say "building…"**
The app is being compiled automatically — check back in about 15 minutes.

**Windows says the app is unsafe**
This is because the app isn't signed with an expensive certificate. It's safe to run — click **More info** → **Run anyway**.

**Mac says it can't be opened because it's from an unidentified developer**
Right-click (or Control-click) the app → **Open** → **Open**. You only need to do this once.

**The voice widget isn't working**
Make sure your browser has permission to use your microphone. Look for a microphone icon in your browser's address bar and allow it.
